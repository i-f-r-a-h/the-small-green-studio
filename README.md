## Getting Started
To get a local copy up and running follow these simple example steps.

### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```
* gulp (see [documentation](https://github.com/gulpjs/gulp))
  ```sh
  npm i gulp
  ```

### Installation
1. Make sure you have Npm and Gulp working
2. Clone the repo
   ```sh
   git clone https://github.com/i-f-r-a-h/the-small-green-studio.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run gulp to build the environment
   ```js
   gulp
   ```
